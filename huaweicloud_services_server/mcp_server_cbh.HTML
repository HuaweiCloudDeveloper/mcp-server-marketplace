简介

云堡垒机（Cloud Bastion Host，CBH）是华为云的一款统一安全管控平台，为企业提供集中的账号（Account）、授权（Authorization）、认证（Authentication）和审计（Audit）管理服务。 云堡垒机提供云计算安全管控的系统和组件，包含部门、用户、资源、策略、运维、审计等功能模块，集单点登录、统一资产管理、多终端访问协议、文件传输、会话协同等功能于一体。通过统一运维登录入口，基于协议正向代理技术和远程访问隔离技术，实现对服务器、云主机、数据库、应用系统等云上资源的集中管理和运维审计。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构，显著扩展Agent的能力边界与协作效率。该协议支持企业快速接入多类服务并实现统一调 用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd mcp-server/tree/master-dev/huaweicloud_services_server/mcp_server_ecs

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-ecs --transport stdio --port 8009

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-ecs --transport http  --port 8009

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-ecs --transport sse --port 8009

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-ecs": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run",
                "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_cbh/src/mcp_server_cbh/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-ecs": {
              "type": "http",
              "url": "http://localhost:8009/mcp" 
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-ecs": {
              "type": "sse",
              "url": "http://localhost:8009/sse"
            }
          }
        }

API工具列表

DDM实例管理

  • UpdateInstanceSecurityGroup
    修改DDM实例安全组。

云堡垒机信息查询

  • SearchQuota
    查询云堡垒机配额信息。
  • ShowAvailableZoneInfo
    获取云堡垒机服务可用分区信息。
  • ShowNetworkConfiguration
    检查云堡垒机实例网络信息。
  • ListCbhInstance
    获取当前租户下的云堡垒机实例列表。
  • ListQuotaStatus
    获取当前租户所选择的可用分区、性能规格所对应的弹性云服务器是否可用。

云堡垒机管理

  • StopCbhInstance
    关闭云堡垒机实例。
  • ResetLoginMethod
    重置admin用户多因子认证方式。
  • UpgradeCbhInstance
    升级云堡垒机实例
  • InstallCbhEip
    云堡垒机实例绑定弹性公网IP
  • UninstallCbhEip
    云堡垒机实例解绑弹性公网IP。
  • StartCbhInstance
    启动云堡垒机实例。
  • RestartCbhInstance
    重启云堡垒机实例。
  • ChangeInstanceNetwork
    修改云堡垒机实例网络。

其他接口

  • ListAvailableZones
    在创建实例时,需要配置实例所在的可用区ID,可通过该接口查询可用区的ID。

委托授权

  • ShowAuthorization
    获取租户给云堡垒机服务委托授权信息。
  • RegisterAuthorization
    租户创建或取消云堡垒机服务的委托授权。

实例管理

  • StopInstance
    实例进行关机,通过暂时停止按需实例以节省费用,实例默认停止七天。
  • ResetPassword
    重置密码(只针对开通SSL的实例)。

操作管理

  • ResetInstancePassword
    重置云堡垒机实例web登录admin用户密码。
  • LoginInstanceAdmin
    用户登录堡垒机实例admin的console。
  • LoginInstance
    IAM用户登录堡垒机实例console。
  • RebootInstance
    重启云堡垒机实例。
  • ShowOmUrl
    获取运维链接
  • ResetInstanceLoginMethod
    重置堡垒机实例admin用户登录方式。
  • RollbackInstance
    回退升级的云堡垒机实例。
  • ChangeInstanceType
    修改单机堡垒机实例类型。

数据库管理

  • StartInstance
    启动数据库,同时支持节点级别的启动操作

标签管理

  • ListInstancesByTag
    使用标签过滤实例。
  • BatchCreateInstanceTag
    操作堡垒机实例资源标签。
  • ShowInstanceTags
    查询指定实例的标签信息。
  • CountInstancesByTag
    统计符合标签条件的实例数量。

流水线管理

  • ShowInstanceStatus
    检查流水线创建状态

独享实例管理

  • CreateInstance
    创建WAF独享引擎实例。独享模式支持的局点包括:华东-青岛、中东-利雅得、华北-北京一、华北-北京四、华北-乌兰察布一、华东-上海一、华东-上海二、华南-广州、华南-深圳、中国-香港、西南-贵阳一、亚太-曼谷、 亚太-新加坡、非洲约翰内斯堡、土耳其-伊斯坦布尔;普通租户类独享支持的局点:华北-北京四、华东-上海一、华南-广州、中国-香港、亚太-曼谷、 亚太-新加坡。
  • DeleteInstance
    删除WAF独享引擎信息。独享模式只在部分局点支持,包括:华北-北京四、华东-上海一、华南-广州、华南-深圳 、中国-香港、亚太-曼谷、 亚太-新加坡。

生命周期管理

  • UpgradeInstance
    实例内核升级
  • ListInstances
    查询租户的实例列表,支持按照条件查询。

网络管理

  • InstallInstanceEip
    云堡垒机实例绑定弹性公网IP。
  • UninstallInstanceEip
    为云堡垒机实例解绑弹性公网IP。
  • SwitchInstanceVpc
    切换堡垒机虚拟私有云

获取IAM登录实例链接

  • LoginCbh
    获取当前IAM用户登录堡垒机的免登录链接

规格变更管理

  • ResizeInstance
    实例规格变更。

规格管理

  • ListSpecifications
    查询云堡垒机规格信息。

订单管理

  • CreateCbh
    创建云堡垒机实例。(创建云堡垒机实例订单前,先调用此接口)
  • CreateInstanceOrder
    创建云堡垒机实例订单。(调用此接口前先调用创建云堡垒机实例接口)
  • ChangeInstanceOrder
    创建变更云堡垒机实例订单。(调用此接口前先调用创建变更云堡垒机实例任务接口,当前接口未开放)

配额

  • ShowQuota
    查询单租户在VPC服务下的网络资源配额,包括vpc配额、子网配额、安全组配额、安全组规则配额、弹性公网IP配额,vpn配额等。

配额管理

  • ShowEcsQuota
    获取当前租户所选择的可用分区里的堡垒机ECS规格是否可用。

镜像标签

  • ListTags
    根据不同条件查询镜像标签列表信息。

以及更多用于元数据、可用区、磁盘管理和监控的API。
