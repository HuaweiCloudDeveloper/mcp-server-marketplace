简介

华为MCP服务器是基于华为云服务构建的模型上下文协议（MCP）服务器，为大型AI模型提供安全可控的云访问能力

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-dis --transport stdio --port 8842

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-dis --transport http  --port 8842

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-dis --transport sse --port 8842

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-dis": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run","/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_dis/src/mcp_server_dis/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-dis": {
              "type": "http",
              "url": "http://localhost:8842/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-dis": {
              "type": "sse",
              "url": "http://localhost:8842/sse"
            }
          }
        }

API工具列表

App管理

  • ListApp
    本接口用于查询APP列表。
  • ShowApp
    本接口用于查询APP详情。
  • CreateApp
    本接口用于创建消费APP。
  • DeleteApp
    本接口用于删除App。
  • ShowConsumerState
    本接口用于查询APP消费状态。

Checkpoint管理

  • CommitCheckpoint
    本接口用于提交Checkpoint。
  • DeleteCheckpoint
    本接口用于删除Checkpoint。
  • ShowCheckpoint
    本接口用于查询Checkpoint详情。

数据管理

  • SendRecords
    本接口用于上传数据到DIS通道中。
  • ConsumeRecords
    本接口用于从DIS通道中下载数据。
  • ShowCursor
    本接口用于获取数据游标。

标签管理

  • BatchDeleteTags
    该接口用于批量删除资源(通道等)标签。此接口为幂等接口:删除时,如果删除的标签不存在,默认处理成功;删除时不对标签字符集范围做校验。删除时tags结构体不能缺失,key不能为空,或者空字符串。
  • ShowStreamTags
    该接口用于查询指定通道的标签信息。
  • CreateTag
    本接口用于给指定通道添加标签。
  • DeleteTag
    该接口用于删除指定通道的标签。
  • ListResourcesByTags
    该接口用于使用标签过滤资源(通道等)。
  • ListTags
    该接口用于查询指定区域所有标签集合。
  • BatchCreateTags
    该接口用于批量添加资源(通道等)标签。此接口为幂等接口:创建时如果请求体中存在重复key则报错。创建时,不允许设置重复key数据,如果数据库已存在该key,就覆盖value的值。

监控管理

  • ShowPartitionMetrics
    本接口用于查询通道指定分区的监控数据。
  • ShowStreamMetrics
    本接口用于查询指定通道的监控数据。

转储任务管理

  • BatchStartTransferTask
    此接口用于批量启动转储任务。
  • DeleteTransferTask
    该接口用于删除转储任务。
  • ShowTransferTask
    查询转储任务详情。
  • BatchStopTransferTask
    此接口用于批量暂停转储任务。
  • CreateDliTransferTask
    本接口用于添加DLI转储任务。
  • CreateObsTransferTask
    本接口用于添加OBS转储任务。
  • ListTransferTasks
    本接口用于查询转储任务列表。
  • CreateMrsTransferTask
    本接口用于添加MRS转储任务。
  • CreateCloudTableTransferTask
    本接口用于添加CloudTable转储任务。
  • CreateDwsTransferTask
    本接口用于添加DWS转储任务。

通道管理

  • CreatePolicies
    本接口用于给指定通道添加权限策略。
  • ShowStream
    本接口用于查询指定通道的详情。
  • ListStreams
    本接口用户查询当前租户创建的所有通道。
  • UpdatePartitionCount
    本接口用于变更指定通道中的分区数量。
  • ListPolicies
    本接口用于查询指定通道的权限策略列表。
  • CreateStream
    本接口用于创建通道。
  • DeleteStream
    本接口用于删除指定通道。
  • UpdateStream
    本接口用于更新指定通道的通道信息。

以及更多用于元数据、可用区、磁盘管理和监控的API.
