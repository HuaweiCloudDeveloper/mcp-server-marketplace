简介

OCR MCP Server 基于自然语言指令，实现图文识别的智能调用，支持通用文字、证件、票据及行业专用文档的自动化识别与结构化提取。无缝对接OCR开放API，助力用户快速构建智能化数据采集系统，提升业务处理效率，广泛适用于办公自动化、金融、物流、医疗等场景。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/ocr-mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-ocr --transport stdio --port 8902

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-ocr --transport http  --port 8902

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-ocr --transport sse --port 8902

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code + Cline

客户端通信配置示例

1. stdio 模式

{
  "mcpServers": {
    "stdio-ocr": {
      "type": "stdio",
      "command": "uv",
      "args": [
        "run", 
        "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_ocr/src/mcp_server_ocr/run.py"
      ]
    }
  }
}

2. http 模式

{
  "mcpServers": {
    "http-ocr": {
      "type": "http",
      "url": "http://localhost:8902/mcp"
    }
  }
}

3. sse 模式

{
  "mcpServers": {
    "sse-ocr": {
      "type": "sse",
      "url": "http://localhost:8902/sse"
    }
  }
}
