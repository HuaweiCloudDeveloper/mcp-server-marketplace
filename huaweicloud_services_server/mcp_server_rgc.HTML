简介

资源治理中心（Resource Governance Center，简称RGC）服务为用户提供搭建安全、可扩展的多账号环境并持续治理的能力。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构， 显著扩展Agent的能力边界与协作效率。该协议支持企业快速接入多类服务并实现统一调用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-rgc --transport stdio --port 8909

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-rgc --transport http  --port 8909

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-rgc --transport sse --port 8909

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-rgc": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run","/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_rgc/src/mcp_server_rgc/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-rgc": {
              "type": "http",
              "url": "http://localhost:8888/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-rgc": {
              "type": "sse",
              "url": "http://localhost:8888/sse"
            }
          }
        }

API工具列表

Governance

  • DisableControl
    关闭组织下的某个单元的某个控制策略。
  • ListControlsForOrganizationalUnit
    列出组织里某个注册OU开启的所有控制策略信息。
  • ShowControlOperate
    根据操作ID查询返回指定ID的操作状态。
  • EnableControl
    给组织下的某个单元开启某个控制策略。
  • ListEnabledControls
    列出组织里开启的所有控制策略信息。

ManagedOrganization

  • RegisterOrganizationalUnit
    将组织里的某个OU注册到RGC服务。
  • ShowOperation
    查询在RGC服务里注册/取消注册的过程信息。
  • ShowManagedAccount
    查询组织里某个纳管账号信息。
  • CreateAccount
    在组织里的某个注册OU下创建账号。

以及更多用于元数据、可用区、磁盘管理和监控的API.
