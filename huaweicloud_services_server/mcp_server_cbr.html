简介

CBR MCP Server 是一款模型上下文协议（Model Context Protocol）服务器，为 MCP 客户端（如 Claude Desktop、Cline、Cursor）提供与华为云服务 CBR 交互的能力。CBR资源的全链条管理可以基于自然语言进行。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp_server_cbr --transport stdio --port 8010

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp_server_cbr --transport http  --port 8010

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp_server_cbr --transport sse --port 8010

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-vpc": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run",
                "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_cbr/src/mcp_server_cbr/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-vpc": {
              "type": "http",
              "url": "http://localhost:8010/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-vpc": {
              "type": "sse",
              "url": "http://localhost:8010/sse"
            }
          }
        }

API工具列表

Checkpoint管理

  • ShowCheckpoint
    本接口用于查询Checkpoint详情。

主机管理

  • ListPolicies
    查询主机策略列表

任务

  • ListOpLogs
    查询任务列表
  • ShowOpLog
    根据指定任务ID查询任务

可保护性

  • ShowReplicationCapabilities
    查询本区域的复制能力
  • ListProtectable
    查询可保护性资源列表
  • CheckAgent
    检查应用一致性Agent状态
  • ShowProtectable
    根据ID查询可保护性资源

后端云服务器

  • DeleteMember
    删除后端云服务器

备份

  • ShowBackup
    根据指定id查询单个副本。
  • ImportBackup
    同步线下混合云VMware备份副本
  • CopyBackup
    跨区域复制备份。
  • UpdateBackup
    根据备份id更改备份
  • RestoreBackup
    恢复备份数据

备份与恢复

  • ListBackups
    获取备份列表。

备份共享

  • AddMember
    添加备份可共享的成员,只有云服务器备份可以添加备份共享成员,且仅支持在同一区域的不同用户间共享。
  • ShowMembersDetail
    获取备份共享成员的列表信息
  • UpdateMemberStatus
    更新备份共享成员的状态,需要接收方执行此API。
  • ShowMemberDetail
    获取备份成员的详情

备份管理API

  • DeleteBackup
    删除备份。

存储库

  • ShowVault
    根据ID查询指定存储库
  • UpdateVault
    根据存储库ID修改存储库
  • ListVault
    查询存储库列表
  • DisassociateVaultPolicy
    存储库解除策略
  • MigrateVaultResource
    支持资源迁移到另一个存储库,不删除备份。
  • BatchUpdateVault
    批量修改项目下所有存储库
  • DeleteVault
    删除存储库。若删除储存库,将一并删除存储库中的所有备份。
  • ListExternalVault
    查询其他区域的存储库列表
  • AssociateVaultPolicy
    存储库设置策略
  • CreatePostPaidVault
    创建包周期存储库
  • SetVaultResource
    设置存储库资源是否自动备份
  • ShowSummary
    查询项目下所有存储库的总容量和总使用量
  • RemoveVaultResource
    移除存储库中的资源,若移除资源,将一并删除该资源在保管库中的备份
  • CreateVault
    创建存储库
  • AddVaultResource
    存储库添加资源

文件应用备份

  • UpdateAgent
    修改客户端状态
  • UnregisterAgent
    移除客户端,移除客户端时将会删除该客户端所有备份,请谨慎操作。
  • RegisterAgent
    注册客户端,安装时候由Agent调用,无需手动注册。
  • ListAgent
    查询客户端列表
  • RemoveAgentPath
    移除已添加的文件备份路径。
  • ShowAgent
    查询指定客户端
  • AddAgentPath
    对客户端新增备份路径,新增的路径不会校验是否存在。

标签

  • DeleteVaultTag
    幂等接口:删除时,如果删除的标签不存在,返回404。Key不能为空或者空字符串。
  • ShowVaultTag
    查询指定实例的标签信息
  • CreateVaultTags
    一个资源上最多有10个标签。
  • ShowVaultResourceInstances
    使用标签过滤实例
  • ShowVaultProjectTag
    查询租户在指定Region和实例类型的所有标签集合
  • BatchCreateAndDeleteVaultTags
    为指定实例批量添加或删除标签

组织策略

  • DeleteOrganizationPolicy
    删除组织策略
  • ListOrganizationPolicyDetail
    查询组织策略每个账号下策略部署状态列表
  • ShowOrganizationPolicy
    查询指定组织策略
  • UpdateOrganizationPolicy
    更新组织策略
  • ListOrganizationPolicies
    查询组织策略列表
  • CreateOrganizationPolicy
    创建组织策略

联邦身份认证管理

  • ShowMetadata
    该接口可以用于[管理员](https://support.huaweicloud.com/usermanual-iam/iam_01_0001.html)查询身份提供商导入到IAM中的Metadata文件。

计量

  • ShowStorageUsage
    查询容量统计

运营

  • ChangeVaultChargeMode
    修改资源的付费模式,暂时只支持按需资源转包周期资源。
  • UpdateOrder
    订单更新,支付cbc订单后,调用该接口更新包周期产品订单信息。该接口已废弃。
  • ChangeOrder
    订单更新,调用该接口更新包周期产品订单信息,返回待支付订单信息。

还原点

  • CreateCheckpoint
    对存储库执行备份,生成备份还原点
  • ImportCheckpoint
    针对vault同步备份副本
  • CopyCheckpoint
    执行复制

防护策略管理

  • DeletePolicy
    删除防护策略,若策略正在使用,则需要先接解除域名与策略的绑定关系才能删除策略。
  • ShowPolicy
    根据Id查询防护策略
  • UpdatePolicy
    更新防护策略,请求体可只传需要更新的部分
  • CreatePolicy
    创建防护策略,系统会在生成策略时配置一些默认的配置项,如果需要修改策略的默认配置项需要通过调用更新防护策略接口实现

项目

  • ListDomainProjects
    根据指定租户名称查询项目列表。
  • ShowMigrateStatus
    查询迁移结果
  • ShowDomain
    由控制台调用的内部接口,用于仅在查询共享备份时获取源project_id的域名信息。
  • MigrateDomain
    将CSBS/VBS资源迁移到CBR。
  • ListProjects
    查询租户的企业项目信息
