简介

华为MCP服务器是基于华为云服务构建的模型上下文协议（MCP）服务器，为大型AI模型提供安全可控的云访问能力

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-ecs --transport stdio --port 8888

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-ecs --transport http  --port 8888

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-ecs --transport sse --port 8888

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-ecs": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_ecs/src/mcp_server_ecs/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-ecs": {
              "type": "http",
              "url": "http://localhost:8888/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-ecs": {
              "type": "sse",
              "url": "http://localhost:8888/sse"
            }
          }
        }

API工具列表

云服务器操作

  • NovaListServerActions
    查询ECS操作历史
  • NovaShowServerAction
    查询特定ECS操作
  • CreatePostPaidServers
    创建按需付费ECS
  • NovaDeleteServer
    删除ECS实例

服务器组管理

  • CreateServerGroup
    创建ECS组
  • ListServerGroups
    查询ECS组
  • AddServerGroupMember
    添加ECS到组
  • DeleteServerGroup
    删除ECS组

安全管理

  • NovaAssociateSecurityGroup
    添加安全组
  • NovaListServerSecurityGroups
    查询安全组
  • NovaCreateKeypair
    创建SSH密钥对
  • ResetServerPassword
    重置ECS密码

以及更多用于元数据、可用区、磁盘管理和监控的API。
