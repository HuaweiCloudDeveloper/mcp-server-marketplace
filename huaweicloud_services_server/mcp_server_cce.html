简介

CCE MCP Server 通过自然语言指令实现企业级Kubernetes集群的智能管理，支持集群创建、升级、节点管理与弹性伸缩等全生命周期操作。集成高级调度策略、多网络/存储模型及全方位监控能力，适用于云原生应用部署、微服务治理、AI/大数据容器化及多集群资源统一运维等场景，助力企业高效构建高可用、高性能的容器化服务，显著降低运维复杂度。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-cce --transport stdio --port 8012

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-cce --transport http  --port 8012

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-cce --transport sse --port 8012

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-cce": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run", 
                "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_cce/src/mcp_server_cce/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-cce": {
              "type": "http",
              "url": "http://localhost:8012/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-cce": {
              "type": "sse",
              "url": "http://localhost:8012/sse"
            }
          }
        }
