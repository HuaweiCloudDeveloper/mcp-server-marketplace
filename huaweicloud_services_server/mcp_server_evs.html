简介

EVS MCP Server 是一款模型上下文协议（Model Context Protocol）服务器，为 MCP 客户端（如 Claude Desktop、Cline、Cursor）提供与华为云服务 EVS 交互的能力。EVS资源的全链条管理可以基于自然语言进行。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp_server_evs.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp_server_evs --transport stdio --port 8859

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp_server_evs --transport http  --port 8859

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp_server_evs --transport sse --port 8859

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-evs": {
              "type": "stdio",
              "command": "uv",
              "args": [
               "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_evs/src/mcp_server_evs/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-evs": {
              "type": "http",
              "url": "http://localhost:8859/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-evs": {
              "type": "sse",
              "url": "http://localhost:8859/sse"
            }
          }
        }

API工具列表

硬盘

  • RetypeVolume
    对按需或者包周期云硬盘进行磁盘类型变更。
  • ListVolumes
    查询所有云硬盘的详细信息。
  • CinderListAvailabilityZones
    查询所有的可用分区信息。
  • ResizeVolume
    对按需或者包周期云硬盘进行扩容。
  • DeleteVolume
    删除一个云硬盘。
  • UpdateVolume
    更新一个云硬盘的名称和描述。
  • UnsubscribePostpaidVolume
    退订包周期计费模式的云硬盘,有如下约束:
  • ModifyVolumeQoS
    调整云硬盘的iops或者吞吐量。
  • CreateVolume
    创建按需或包周期云硬盘。
  • CinderListQuotas
    查询租户的详细配额。
  • CinderListVolumeTypes
    查询云硬盘类型列表。

云硬盘快照

  • RollbackSnapshot
    将快照数据回滚到云硬盘。支持企业项目授权功能。
  • ListSnapshots
    查询云硬盘快照详细列表信息。
  • DeleteSnapshot
    删除云硬盘快照。
  • CreateSnapshot
    创建云硬盘快照。
  • ShowSnapshot
    查询单个云硬盘快照信息。支持企业项目授权功能。
  • UpdateSnapshot
    更新云硬盘快照。支持企业项目授权功能。

云硬盘标签

  • ShowVolumeTags
    查询指定云硬盘的标签信息。
  • ListVolumeTags
    获取某个租户的所有云硬盘资源的标签信息。
  • ListVolumesByTags
    通过标签查询云硬盘资源实例详情。
  • BatchCreateVolumeTags
    为指定云硬盘批量添加标签。
  • BatchDeleteVolumeTags
    为指定云硬盘批量删除标签。

云硬盘过户

  • CinderDeleteVolumeTransfer
    当云硬盘过户未被接受时,您可以删除云硬盘过户记录,接受后则无法执行删除操作。
  • CinderListVolumeTransfers
    查询当前租户下所有云硬盘的过户记录列表
  • CinderCreateVolumeTransfer
    指定云硬盘来创建云硬盘过户记录,创建成功后,会返回过户记录ID以及身份认证密钥。
  • CinderShowVolumeTransfer
    查询单个云硬盘的过户记录详情,比如过户记录创建时间、ID以及名称等信息。
  • CinderAcceptVolumeTransfer
    通过云硬盘过户记录ID以及身份认证密钥来接受云硬盘过户。

查询密钥API版本信息

  • ShowVersion
    - 功能介绍:查指定API版本信息。
  • ListVersions
    查询SMN开放API支持的版本号。

边缘硬盘

  • ShowVolume
    查询硬盘详情。

镜像任务

  • ShowJob
    该接口为扩展接口,主要用于查询异步接口执行情况,比如查询导出镜像任务的执行状态。
