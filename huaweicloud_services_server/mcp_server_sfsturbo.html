简介

高性能弹性文件服务（Scalable File Service Turbo，SFS Turbo）提供按需扩展的高性能文件存储（NAS），可为云上多个弹性云服务器（Elastic Cloud Server，ECS）、容器（CCE&CCI）、裸金属服务器（BMS）提供共享访问。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构， 显著扩展Agent的能力边界与协作效率。该协议支持企业快速接入多类服务并实现统一调用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp_server_sfsturbo --transport stdio --port 8916

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp_server_sfsturbo --transport http  --port 8916

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp_server_sfsturbo --transport sse --port 8916

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-vpc": {
              "type": "stdio",
              "command": "uv",
              "args": [
               "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_sfsturbo/src/mcp_server_sfsturbo/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-vpc": {
              "type": "http",
              "url": "http://localhost:8916/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-vpc": {
              "type": "sse",
              "url": "http://localhost:8916/sse"
            }
          }
        }

API工具列表

任务管理

  • ShowJobDetail
    查询job的执行状态。 可用于查询SFS Turbo异步API的执行状态。例如:可使用调用创建并绑定ldap配置接口时返回的jobId,通过该接口查询job的执行状态。

共享标签

  • ListSharedTags
    查询租户所有共享的标签集合。
  • ShowSharedTags
    查询指定共享的所有标签信息。
  • CreateSharedTag
    指定共享添加一个标签。
  • BatchAddSharedTags
    指定共享批量添加标签。
  • DeleteSharedTag
    指定共享删除一个标签。当共享中不存在指定要删除的key时,接口调用将会返回404错误。
  • ListSharesByTag
    通过标签查询文件系统列表

名称管理

  • ChangeShareName
    修改文件系统名称

存储联动管理

  • ListHpcCacheTasks
    查询数据导入导出任务列表
  • UpdateObsTargetAttributes
    更新后端存储属性
  • DeleteHpcCacheTask
    删除数据导入导出任务
  • DeleteBackendTarget
    删除后端存储
  • CreateHpcCacheTask
    创建数据导入导出任务
  • ShowBackendTargetInfo
    获取后端存储详细信息
  • UpdateHpcShare
    更新文件系统冷数据淘汰时间
  • UpdateObsTargetPolicy
    更新后端存储自动同步策略
  • ListBackendTargets
    查询后端存储列表
  • ShowHpcCacheTask
    查询数据导入导出任务详情
  • CreateBackendTarget
    为SFS Turbo 文件系统绑定后端存储

文件系统管理

  • ShowFsTask
    获取文件系统异步任务详情。仅支持查询目录资源使用情况的任务,API请求路径的feature取值为dir-usage,以下简称为DU任务。
  • CreateFsTask
    创建文件系统异步任务,仅支持异步查询目录资源使用情况,API请求路径的feature取值为dir-usage,以下简称为DU任务。
  • SetHpcCacheBackend
    配置hpc缓存型后端信息
  • ListFsTasks
    获取文件系统异步任务列表。仅支持查询目录资源使用情况的任务,API请求路径的feature取值为dir-usage,以下简称为DU任务。
  • DeleteFsTask
    如果异步任务正在执行,则取消并删除任务;否则,删除任务。仅支持删除目录资源使用情况的任务,API请求路径的feature取值为dir-usage,以下简称为DU任务。

  • ShowShare

目录管理

  • UpdateFsDirQuota
    更新目标文件夹quota
  • CreateFsDir
    创建目录
  • ShowFsDirUsage
    查询目录资源使用情况(包括子目录的资源)。后端有5min的缓存时间,查询的数据可能有延迟。

    查询目录是否存在

  • DeleteFsDirQuota

连接管理

  • ChangeSecurityGroup
    修改SFS Turbo文件系统绑定的安全组。修改安全组为异步任务,可以通过“查询单个文件系统”返回的子状态字段“sub_status”来判断是否修改安全组状态,子状态为“232”即为修改安全组成功。
