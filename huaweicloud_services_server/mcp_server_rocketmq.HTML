简介

华为MCP服务器是基于华为云服务构建的模型上下文协议（MCP）服务器，为大型AI模型提供安全可控的云访问能力

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-rocketmq --transport stdio --port 8888

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-rocketmq --transport http  --port 8888

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-rocketmq --transport sse --port 8888

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-rocketmq": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run","/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_rocketmq/src/mcp_server_rocketmq/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-ecs": {
              "type": "http",
              "url": "http://localhost:8888/mcp"
               
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-ecs": {
              "type": "sse",
              "url": "http://localhost:8888/sse"
            }
          }
        }

API工具列表

Topic管理

  • UpdateTopic
    修改主题。
  • ListConsumerGroupOfTopic
    查询主题消费组列表。
  • DeleteTopic
    删除指定主题。
  • ShowTopicStatus
    查询主题的消息数。
  • CreateTopicOrBatchDeleteTopic
    创建主题或批量删除主题。
  • ShowOneTopic
    查询单个主题。
  • ListRocketInstanceTopics
    该接口用于查询指定RocketMQ实例的Topic列表。

元数据迁移

  • ListRocketMqMigrationTask
    1. 查询实例下所有迁移任务
  • CreateRocketMqMigrationTask
    新建元数据迁移任务。
  • DeleteRocketMqMigrationTask
    删除元数据迁移任务。

其他接口

  • ListEngineProducts
    查询相应引擎的产品规格列表。
  • ListAvailableZones
    在创建实例时,需要配置实例所在的可用区ID,可通过该接口查询可用区的ID。
  • ShowRocketMQProductCores
    查询RocketMQ产品规格核数。

参数管理

  • UpdateRocketMqConfigs
    该接口用于修改RocketMQ配置。
  • ShowRocketMqConfigs
    该接口用于查询RocketMQ配置,若成功则返回配置的相关信息。

实例管理

  • ShowInstanceNodes
    查询实例节点信息。
  • RestartInstance
    重启指定实例。

标签管理

  • ShowRocketmqProjectTags
    查询项目标签。
  • BatchCreateOrDeleteRocketmqTag
    批量添加或删除实例标签。
  • ShowRocketmqTags
    查询实例标签。

消息管理

  • ExportDlqMessage
    导出死信消息。
  • ListMessages
    查询消息。
  • SendDlqMessage
    重发死信消息。
  • ListMessageTrace
    查询消息轨迹。
  • ValidateConsumedMessage
    消费验证。

消费组管理

  • UpdateConsumerGroup
    修改指定消费组参数。
  • ShowConsumerListOrDetails
    查询消费列表或详情。
  • ListInstanceConsumerGroups
    查询消费组列表。
  • ShowConsumerConnections
    查询消费组内消费者列表
  • DeleteConsumerGroup
    删除指定消费组。
  • CreateConsumerGroupOrBatchDeleteConsumerGroup
    创建消费组或批量删除消费组。
  • ResetConsumeOffset
    重置消费进度。
  • BatchUpdateConsumerGroup
    批量修改消费组。
  • ShowGroup
    查询指定消费组详情。

生命周期管理

  • ListBrokers
    查询代理列表。
  • EnableDns
    开启RocketMQ实例域名访问能力。
  • DeleteInstance
    删除指定的实例,释放该实例的所有资源。
  • UpdateInstance
    修改实例的名称和描述信息。
  • CreateInstanceByEngine
    创建实例,该接口支持创建按需和包周期两种计费方式的实例。
  • BatchDeleteInstances
    批量删除实例。**实例删除后,实例中原有的数据将被删除,且没有备份,请谨慎操作。**
  • ShowInstance
    查询指定实例的详细信息。
  • ListInstances
    查询租户的实例列表,支持按照条件查询。

用户管理

  • ListUser
    查询用户列表。
  • UpdateUser
    修改用户参数。
  • ShowUser
    查询用户详情。
  • ListTopicAccessPolicy
    查询主题的授权用户列表。
  • CreateUser
    创建用户。
  • ListConsumeGroupAccessPolicy
    查询消费组的授权用户列表。
  • DeleteUser
    删除用户。

规格变更管理

  • ResizeInstance
    实例规格变更。
  • ShowEngineInstanceExtendProductInfo
    查询实例的扩容规格列表。

以及更多用于元数据、可用区、磁盘管理和监控的API.
