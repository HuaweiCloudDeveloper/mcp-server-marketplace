简介

企业路由器（Enterprise Router, ER）可以连接虚拟私有云（Virtual Private Cloud, VPC）或本地网络来构建中心辐射型组网，是云上大规格、高带宽、高性能的集中路由器。 企业路由器使用边界网关协议（Border Gateway Protocol, BGP），支持路由学习、动态选路以及链路切换，极大的提升网络的可扩展性及运维效率，从而保证业务的连续性。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构， 显著扩展Agent的能力边界与协作效率。该协议支持企业快速接入多类服务并实现统一调 用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp_server_er --transport stdio --port 8858

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp_server_er --transport http  --port 8858

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp_server_er --transport sse --port 8858

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-er": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_er/src/mcp_server_er/run.py"
              ]
            }
          }
        }
2. http 模式 
{
          "mcpServers": {
            "http-er": {
              "type": "http",
              "url": "http://localhost:8858/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-er": {
              "type": "sse",
              "url": "http://localhost:8858/sse"
            }
          }
        }

API工具列表

Association

  • ListAssociations
    查询路由关联列表。

Attachments

  • ListAttachments
    查询企业路由器实例下的连接列表。
  • ShowAttachment
    查询连接详情
  • UpdateAttachment
    修改连接基本信息。
  • RejectAttachment
    拒绝共享连接创建
  • AcceptAttachment
    接受共享连接创建

AvailableZone

  • ListAvailabilityZone
    查询支持创建企业路由器实例的可用区列表,当可用区状态为available时,表示可以创建企业路由器实例。

EnterpriseRouterInstance

  • ChangeAvailabilityZone
    更新企业路由器的可用区信息,企业路由器实例状态为available的时候才能更新。
  • ListEnterpriseRouters
    查询企业路由器列表
  • CreateEnterpriseRouter
    创建企业路由器实例,如果使能默认关联路由表或使能默认传递路由表,那么系统会默认创建一张路由表,作为默认关联路由表或默认传递路由表。
  • UpdateEnterpriseRouter
    更新企业路由器基本信息。
  • ShowEnterpriseRouter
    查询企业路由器详情
  • DeleteEnterpriseRouter
    删除企业路由器。

FlowLog

  • EnableFlowLog
    开启流日志
  • DisableFlowLog
    关闭流日志

Propagation

  • ListPropagations
    查询路由传播列表。
  • DisablePropagation
    解绑连接和路由表的传播关系。
  • EnablePropagation
    每个连接可以和多个路由表建立传播关系,从该连接学习到的路由会应用到具有传播关系的路由表。

Route

  • ListStaticRoutes
    查询静态路由列表。
  • ListEffectiveRoutes
    查询有效的路由列表,支持分页查询能力
  • CreateStaticRoute
    创建静态路由
  • UpdateStaticRoute
    更新静态路由
  • ShowStaticRoute
    查询静态路由详情
  • DeleteStaticRoute
    删除静态路由

Tags

  • ListProjectTags
    查询租户在指定项目中指定资源类型下的所有标签
  • BatchCreateResourceTags
    为指定实例批量添加标签

VPCAttachment

  • ListVpcAttachments
    查询企业路由器实例下的VPC连接列表。
  • ShowVpcAttachment
    查询VPC连接详情
  • UpdateVpcAttachment
    修改VPC连接基本信息。
  • CreateVpcAttachment
    给ER实例创建VPC连接。
  • DeleteVpcAttachment
    删除VPC连接。

流日志

  • ShowFlowLog
    查询流日志详情
  • UpdateFlowLog
    更新流日志
  • DeleteFlowLog
    删除流日志
  • ListFlowLogs
    查询提交请求的租户的所有流日志列表,并根据过滤条件进行过滤
  • CreateFlowLog
    创建流日志。

资源标签

  • ShowResourceTag
    查询单个资源上的标签。
  • CreateResourceTag
    用于给云服务的多个资源添加标签,每个资源最多可添加10个标签,每次最多支持批量操作20个资源。
  • DeleteResourceTag
    用于批量移除云服务多个资源的标签,每个资源最多支持移除10个标签,每次最多支持批量操作20个资源。

路由表

  • ListRouteTables
    查询提交请求的帐户的所有路由表列表,并根据过滤条件进行过滤
  • DisassociateRouteTable
    子网解关联路由表
  • CreateRouteTable
    创建路由表
  • DeleteRouteTable
    删除路由表
  • AssociateRouteTable
    路由表关联子网。子网关联路由表A后,再关联B,不需要先跟路由表A解关联再关联路由表B
  • ShowRouteTable
    查询路由表详情
  • UpdateRouteTable
    更新路由表,包括可以更新路由表的名称,描述,以及新增、更新、删除路由条目

配额

  • ShowQuotas
    查询当前项目下资源配额情况。
