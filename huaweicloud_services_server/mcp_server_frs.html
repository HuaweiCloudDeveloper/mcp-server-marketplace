简介

GES MCP Server 基于自然语言指令，实现图引擎服务的智能管控，支持图创建、查询、分析与可视化操作。内置丰富图算法，兼容Gremlin查询语言，提供可视化探索环境与API接口，适用于社交分析、推荐系统、反欺诈等场景，助力用户高效挖掘关系数据价值。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/frs-mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-frs --transport stdio --port 8862

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-frs --transport http  --port 8862

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-frs --transport sse --port 8862

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code + Cline

客户端通信配置示例

1. stdio 模式

{
  "mcpServers": {
    "stdio-frs": {
      "type": "stdio",
      "command": "uv",
      "args": [
        "run", 
        "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_frs/src/mcp_server_frs/run.py"
      ]
    }
  }
}

2. http 模式

{
  "mcpServers": {
    "http-frs": {
      "type": "http",
      "url": "http://localhost:8862/mcp"
    }
  }
}

3. sse 模式 - SSE 传输已弃用，可能会在未来版本中移除

{
  "mcpServers": {
    "sse-frs": {
      "type": "sse",
      "url": "http://localhost:8862/sse" 
    }
  }
}

API工具列表

人脸库资源管理

  • ShowAllFaceSets
    查询当前用户所有人脸库的状态信息。
  • ShowFaceSet
    查询人脸库当前的状态。
  • DeleteFaceSet
    删除人脸库以及其中所有的人脸。
  • CreateFaceSet
    创建用于存储人脸特征的人脸库。

人脸搜索

  • SearchFaceByFaceId
    人脸搜索是指在已有的人脸库中,查询与目标人脸相似的一张或者多张人脸,并返回相应的置信度。
  • SearchFaceByBase64
    同上，输入为Base64编码。
  • SearchFaceByUrl
    同上，输入为图片URL。
  • SearchFaceByFile
    同上，输入为本地文件。

人脸检测

  • DetectFaceByFile
    对输入图片进行人脸检测和分析,输出人脸位置、关键点及属性。
  • DetectFaceByUrl
    同上，输入为图片URL。
  • DetectFaceByBase64
    同上，输入为Base64编码。

人脸比对

  • CompareFaceByFile
    将两个人脸进行比对,判断是否为同一个人,返回比对置信度。
  • CompareFaceByUrl
    同上，输入为图片URL。
  • CompareFaceByBase64
    同上，输入为Base64编码。

人脸资源管理

  • UpdateFace
    根据人脸ID(face_id)更新单张人脸信息。
  • AddFacesByBase64
    添加人脸到人脸库中，支持最大人脸或所有人脸。
  • AddFacesByFile
    同上，输入为本地文件。
  • ShowFacesByLimit
    查询指定人脸库中人脸信息。
  • DeleteFaceByFaceId
    根据face_id删除人脸。
  • DeleteFaceByExternalImageId
    根据external_image_id删除人脸。
  • ShowFacesByFaceId
    查询指定人脸库中人脸信息。
  • AddFacesByUrl
    添加人脸到人脸库中，支持最大人脸或所有人脸。
  • BatchDeleteFaces
    自定义筛选条件,批量删除符合指定条件的多张人脸。

动作活体检测

  • DetectLiveByFile
    通过判断视频中的人物动作是否一致来识别是否为活体。
  • DetectLiveByUrl
    同上，输入为图片URL。
  • DetectLiveByBase64
    同上，输入为Base64编码。

静默活体检测

  • DetectLiveFaceByFile
    基于图像信息判断是否为真人活体，有效抵御翻拍攻击。
  • DetectLiveFaceByUrl
    同上，输入为图片URL。
  • DetectLiveFaceByBase64
    同上，输入为Base64编码。

以及更多用于元数据、可用区、磁盘管理和监控的API。
