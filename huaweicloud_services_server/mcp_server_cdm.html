简介

CDM MCP服务器是一款基于模型上下文协议（Model Context Protocol）的服务端，可为MCP客户端（如Claude Desktop、Cline、Cursor等）提供与华为云服务CDM交互的能力，支持基于自然语言对CDM资源进行全链路管理。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-cdm --transport stdio --port 8015

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-cdm --transport http  --port 8015

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-cdm --transport sse --port 8015

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-cdm": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run", 
                "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_cdm/src/mcp_server_cdm/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-cdm": {
              "type": "http",
              "url": "http://localhost:8015/mcp" 
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-cdm": {
              "type": "sse",
              "url": "http://localhost:8015/sse" 
            }
          }
        }

API工具列表

Job相关接口

  • DeleteJob
    删除Job

    启动Job

作业管理

  • ShowSubmissions
    查询作业执行历史接口
  • CreateAndStartRandomClusterJob
    随机集群创建作业并执行接口
  • StopJob
    在MRS集群中终止指定作业
  • UpdateJob
    更新租户指定ID任务详情
  • CreateJob
    创建单个任务

连接管理

  • UpdateLink
    修改连接接口

    删除连接接口
  • CreateLink
    创建连接接口
  • ShowLink
    查询连接接口

集群管理

  • StartCluster
    启动集群
  • ShowDatastores
    查询CDM集群支持的版本
  • ShowFlavorDetail
    查询指定规格ID的规格详请
  • ModifyCluster
    修改CDM集群配置
  • ShowClusterEnterpriseProjects
    查询指定集群的企业项目ID

    查询当前项目下的所有集群的企业项目ID

  • ShowAvailabilityZones
    查询CDM集群的所有可用区

    查询集群实例信息
  • DeleteCluster
    删除集群服务

    创建一个MRS集群
  • ShowClusterDetail
    查询并显示单个集群详情
  • ListClusters
    查看用户创建的集群列表信息
