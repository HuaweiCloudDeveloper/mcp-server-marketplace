简介

OMS MCP服务器是一款基于模型上下文协议（Model Context Protocol）的服务端，可为MCP客户端（如Claude Desktop、Cline、Cursor等）提供与华为云服务OMS交互的能力，支持基于自然语言对OMS资源进行全链路管理。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-oms --transport stdio --port 8903

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-oms --transport http  --port 8903

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-oms --transport sse --port 8903

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-oms": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run",
                "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_oms/src/mcp_server_oms/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-oms": {
              "type": "http",
              "url": "http://localhost:8903/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-oms": {
              "type": "sse",
              "url": "http://localhost:8903/sse"
            }
          }
        }
