简介

视频直播服务（Live）依托华为云，凭借在视频领域的长期技术积累，快速敏捷地为行业提供高并发、低延迟、超清流畅、安全可靠的端到端音视频直播解决方案。为了更好地服务于不同的直播应用场景，视频直播提供了标准直播与低时延直播两种服务。标 准直播致力于为常规的大规模实时观看场景提供更加稳定、高效的直播流媒体传输服务及功能强大的实时媒体处理能力。低时延直播将直播端到端延迟降低至毫秒级，并且弱网抗性更强，适用于直播延迟敏感，内容同步性要求高的直播场景。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构，显著扩展Agent的能力边界与协作效率。 该协议支持企业快速接入多类服务并实现统一调 用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp_server_das --transport stdio --port 8892

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp_server_das --transport http  --port 8892

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp_server_das --transport sse --port 8892

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-vpc": {
              "type": "stdio",
              "command": "uv",
              "args": [
                 "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_das/src/mcp_server_das/run.py"
              ],
              "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-vpc": {
              "type": "http",
              "url": "http://localhost:8892/mcp",
               "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-vpc": {
              "type": "sse",
              "url": "http://localhost:8892/sse",
              "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }
