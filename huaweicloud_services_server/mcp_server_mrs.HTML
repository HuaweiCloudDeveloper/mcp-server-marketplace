简介

华为MCP服务器是基于华为云服务构建的模型上下文协议（MCP）服务器，为大型AI模型提供安全可控的云访问能力

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-mrs --transport stdio --port 8886

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-mrs --transport http  --port 8886

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-mrs --transport sse --port 8886

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-mrs": {
              "type": "stdio",
              "command": "uv",
              "args": [
                "run","/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_mrs/src/mcp_server_mrs/run.py"
              ],
              "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-mrs": {
              "type": "http",
              "url": "http://localhost:8886/mcp",
               "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-mrs": {
              "type": "sse",
              "url": "http://localhost:8886/sse",
              "env": {
                "HUAWEI_CLOUD_AK": "Your Access Key AK",
                "HUAWEI_CLOUD_SK": "Your Access Key SK"
              }
            }
          }
        }

API工具列表

AvailableZone

  • ListAvailableZones
    在创建集群时,需要配置实例所在的可用区ID,可通过该接口查询可用区的ID。

ClusterManagement

  • ExpandCluster
    对MRS集群进行扩容。
  • ListNodes
    查询集群节点列表。
  • AddComponent
    集群添加组件
  • ShrinkCluster
    对MRS集群进行缩容。

DataConnectorManagement

  • UpdateDataConnector
    更新数据连接
  • ListDataConnector
    查询数据连接列表
  • CreateDataConnector
    创建数据连接
  • DeleteDataConnector
    删除数据连接

IAMSyncManagement

  • UpdateSyncIamUser
    将IAM用户和用户组同步到manager,指定用户的情况下,会将该用户关联的IAM用户组也同步到manager。
  • CancelSyncIamUser
    指定用户、用户组取消同步
  • ShowSyncIamUser
    获取已经同步的IAM用户和用户组

SQL接口

  • CancelSql
    在MRS集群中取消一条SQL的执行任务。
  • ShowSqlResult
    在MRS集群中查询一条SQL的执行结果。
  • ExecuteSql
    在MRS集群中提交并执行一条SQL语句。

TagManagement

  • SwitchClusterTags
    对已有集群启用或关闭集群默认标签。开启后,集群内节点会打上集群默认标签。
  • ShowTagQuota
    查询标签配额信息
  • ShowTagStatus
    查询集群默认标签状态

VersionMetaQuery

  • ShowMrsVersionList
    展示MRS版本列表
  • ShowMrsFlavors
    查询MRS集群版本可用的规格
  • ShowMrsVersionMetadata
    查询对应版本元数据。如果参数里指定集群id,则可查询集群更新过补丁之后的最新元数据。

作业管理接口

  • ShowJobExeListNew
    在MRS指定集群中查询作业列表信息。
  • ShowSingleJobExe
    在MRS集群中查询指定作业的详细信息。
  • ShowSqlResultWithJob
    在MRS集群中查询SparkSql和SparkScript两种类型作业的SQL语句运行完成后返回的查询结果。
  • BatchDeleteJobs
    在MRS集群中批量删除作业。
  • StopJob
    在MRS集群中终止指定作业。
  • CreateExecuteJob
    在MRS集群中新增并提交一个作业。

委托管理

  • UpdateAgencyMapping
    更新用户(组)与IAM委托之间的映射关系。
  • ShowAgencyMapping
    获取用户(组)与IAM委托之间的映射关系的详细信息。

弹性伸缩接口

  • DeleteAutoScalingPolicy
    删除弹性伸缩策略。
  • CreateScalingPolicy
    对弹性伸缩规则进行编辑。
  • UpdateAutoScalingPolicy
    更新弹性伸缩策略。
  • ShowAutoScalingPolicy
    查看指定集群的所有的弹性伸缩策略信息。
  • CreateAutoScalingPolicy
    创建弹性伸缩策略。

标签管理接口

  • CreateClusterTag
    为特定的集群添加一个tag。
  • ListClustersByTags
    使用标签过滤集群。
  • ListClusterTags
    查询指定集群的标签信息。
  • BatchDeleteClusterTags
    为指定集群批量删除标签。
  • ListAllTags
    查询租户在指定Region下的所有标签集合 。
  • DeleteClusterTag
    删除特定集群的标签。
  • BatchCreateClusterTags
    为指定集群批量添加标签。

集群HDFS文件接口

  • ShowHdfsFileList
    在MRS集群中获取指定目录文件列表。

集群管理接口

  • UpdateClusterScaling
    创建集群后,扩容/缩容集群Core节点或者Task节点。MRS集群创建成功后不支持调整Master节点数量,即不支持扩缩容Master节点。该接口不兼容Sahara。
  • RunJobFlow
    创建一个MRS集群并提交作业,并支持作业完成后删除集群,支持MRS 1.8.9及以上集群版本使用。使用接口前,您需要先获取下的资源信息。
  • DeleteCluster
    数据完成处理分析后或者集群运行异常无法提供服务时可删除集群服务。该接口兼容Sahara。
  • ListClusters
    查看用户创建的集群列表信息。该接口不兼容Sahara。
  • ListHosts
    该接口用于查询输入集群的主机列表详情。
  • CreateCluster
    创建一个MRS集群,并在集群中提交一个作业。该接口不兼容Sahara。
  • ShowClusterDetails
    查看指定集群的详细信息。该接口不兼容Sahara。
  • UpdateClusterName
    修改集群名称

以及更多用于元数据、可用区、磁盘管理和监控的API.
