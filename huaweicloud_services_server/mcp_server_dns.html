简介

云解析服务（Domain Name Service，DNS）提供高可用、高扩展的DNS服务，把人们常用的域名（如www.example.com）转换成用于计算机连接的IP地址（如 192.1.2.3）。 云解析服务可以让您直接在浏览器中输入域名，访问网站或Web应用程序。 MCP（Model Context Protocol）为智能体（Agentic）应用提供底层支撑，通过标准化接口与服务集成架构， 显著扩展Agent的能力边界与协作效率。该协议支持企业快速接入多类服务并实现统一调 用，降低系统复杂度和开发成本，助力构建高性能、可扩展的企业级智能体解决方案。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/mcp-server.git

# 进入项目目录
cd /your-path/mcp-server-main

# 设置环境变量 
export HUAWEI_ACCESS_KEY="xxxx"
export HUAWEI_SECRET_KEY="xxxx"

# STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-ecs --transport stdio --port 8846

# 可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-ecs --transport http  --port 8846

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-ecs --transport sse --port 8846

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code+ Cline

客户端通信配置示例

1. stdio 模式

{
          "mcpServers": {
            "stdio-ecs-example": {
              "type": "stdio",
              "command": "uv",
              "args": [
              "run", "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_dns/src/mcp_server_dns/run.py"
              ]
            }
          }
        }

2. http 模式

{
          "mcpServers": {
            "http-ecs-example": {
              "type": "http",
              "url": "http://localhost:8846/mcp"
            }
          }
        }

3. sse 模式-SSE 传输已弃用，可能会在未来版本中移除

{
          "mcpServers": {
            "sse-ecs-example": {
              "type": "sse",
              "url": "http://localhost:8846/sse"
            }
          }
        }
