简介

ImageSearch MCP服务器是一款基于模型上下文协议（Model Context Protocol）的服务端，可为MCP客户端（如Claude Desktop、Cline、Cursor等）提供与华为云服务ImageSearch交互的能力，支持基于自然语言对ImageSearch资源进行全链路管理。

前提条件

  • 云服务账号：准备华为云提供商的访问凭证
  • AK/SK准备：获取华为云提供商的Access Key和Secret Key
  • Python环境：Python 3.12+
  • UV工具：跨平台安装工具

安装UV

Linux/macOS：

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows：

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

使用方法

启动服务

# 克隆仓库
git clone git@github.com:HuaweiCloudDeveloper/imagesearch-mcp-server.git

# 进入项目目录
cd mcp-server/tree/master-dev/huaweicloud_services_server/mcp_server_imagesearch

#STDIO（默认）：最适合本地工具和命令行脚本
uv run mcp-server-imagesearch --transport stdio --port 8876

#可流式传输的 HTTP：推荐用于 Web 部署
uv run mcp-server-imagesearch --transport http  --port 8876

# SSE：为了与现有的 SSE 客户端兼容。
uv run mcp-server-imagesearch --transport sse --port 8876

快速开始

支持以下客户端与MCP服务交互：

  • Cherry Studio（官方客户端）
  • Cursor（官方客户端）
  • Windsurf（官方客户端）
  • VS Code + Cline

客户端通信配置示例

1. stdio 模式

{
  "mcpServers": {
    "stdio-imagesearch": {
      "type": "stdio",
      "command": "uv",
      "args": [
        "run", 
        "/your-path/mcp-server-main/huaweicloud_services_server/mcp_server_imagesearch/src/mcp_server_imagesearch/run.py"
      ],
      "env": {
        "HUAWEI_CLOUD_AK": "Your Access Key AK",
        "HUAWEI_CLOUD_SK": "Your Access Key SK"
      }
    }
  }
}

2. http 模式

{
  "mcpServers": {
    "http-imagesearch": {
      "type": "http",
      "url": "http://localhost:8876/mcp",
      "env": {
        "HUAWEI_CLOUD_AK": "Your Access Key AK",
        "HUAWEI_CLOUD_SK": "Your Access Key SK"
      }
    }
  }
}

3. sse 模式 - SSE 传输已弃用，可能会在未来版本中移除

{
  "mcpServers": {
    "sse-imagesearch": {
      "type": "sse",
      "url": "http://localhost:8876/sse",
      "env": {
        "HUAWEI_CLOUD_AK": "Your Access Key AK",
        "HUAWEI_CLOUD_SK": "Your Access Key SK"
      }
    }
  }
}

API工具列表

删除数据

  • RunDeleteData
    删除指定服务实例中的对应数据,支持指定ID删除和条件删除。

搜索

  • RunSearch
    从指定服务实例中进行搜索。

更新数据

  • RunUpdateData
    更新指定服务实例中的对应数据。

检查数据

  • RunCheckData
    检查指定服务实例中的对应数据,支持指定ID检查和条件检查。

添加数据

  • RunAddData
    添加数据到指定服务实例中。

以及更多用于元数据、可用区、磁盘管理和监控的API。
